# Client: EHP (Environmental Health Products)

# To run the PMF (Performance Management Framework) shiny app

1) Create the folders: ar, as, cs, ea, rc, responses, rm, rn1 in the same location as the ui.R and server.R
2) Run data.R
3) Run helpers.R
4) Run the app
